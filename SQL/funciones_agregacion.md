# Funciones de agregacion
Permiten agrupar datos se utilizan con la clausula `SELECT`, utilizando estas funciones puedes obtener datos como el promedio de un conjunto de valores, el numero de registro de una tabla, el valor maximo o minimo de un campo, y datos estadisticos. La forma en que se utiliza estas funciones no varian mucho de como seria en cualquier otro lenguaje, ejemplo `SELECT nombreDeLaFuncion(campo)`, el valor que reciben es un campo de la tabla

## Algunas funciones de agregacion
- `COUNT(campo)`, cuenta el numero total de valores que posee ese campo
- `SUM(campo)`, suma todos los valores de esa columna
- `AVG(campo)`, obtiene el promedio de todos los valores de esa columna
- `MIN()` Y `MAX()`
