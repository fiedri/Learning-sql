# Que es sql y como puedes crear un base de datos con tablas
SQL (Structured Query Language), desarrollado en 1970, es un lenguaje de programacion usado para el almacenamiento y manejo de datos en base de datos relacionales. Permite agregar, cambiar, borrar, encontrar o recuperar datos proporcionando un conjunto de comandos para consultas, filtrado, ordenacion y agregacion de datos
